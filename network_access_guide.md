
# 網絡訪問與外部訪問配置指南

本指南將說明如何設定本地網絡訪問，並提供從外部（例如在公司或手機上）訪問您部署在 NAS 上的「租屋行情分析系統」的方案。

---

## 1. 本地網絡訪問

在您的家庭或辦公室內部網絡中，訪問系統非常簡單。

-   **Windows 電腦**：
    -   如果您將系統部署在 Windows 電腦上，訪問地址是 `http://127.0.0.1:8000/index_v6.html`。
    -   如果您想讓網絡中的其他設備（例如手機或平板）訪問，請先獲取您電腦的內部 IP 位址（例如 `192.168.1.100`），然後訪問 `http://192.168.1.100:8000/index_v6.html`。

-   **Synology NAS**：
    -   直接使用 NAS 的內部 IP 位址訪問：`http://YOUR_NAS_IP:8000/index_v6.html`。

## 2. 外部訪問規劃

要從外部網絡安全地訪問您家中的 NAS，我們推薦使用 Synology 提供的 **QuickConnect** 服務。這是最簡單、最安全的方法，無需複雜的路由器設定。

### 2.1. 設定 QuickConnect

1.  **登入 DSM**：
    進入您的 NAS 管理介面。

2.  **前往 QuickConnect 設定**：
    -   打開「控制台」。
    -   在「連線能力」區域，點擊「QuickConnect」。

3.  **啟用 QuickConnect**：
    -   勾選「啟用 QuickConnect」。
    -   如果您尚未登入 Synology 帳戶，系統會引導您登入或註冊。
    -   **QuickConnect ID**：設定一個您喜歡且唯一的 ID，例如 `my-rental-analysis`。這將成為您外部訪問的專屬網址。

4.  **設定權限**：
    -   點擊「進階設定」。
    -   確保「DSM」和「檔案分享」等服務已啟用。

### 2.2. 使用 QuickConnect 訪問

設定完成後，您就可以在任何地方，通過以下網址訪問您的 NAS：

```
http://QuickConnect.to/YOUR_QUICKCONNECT_ID
```

例如：`http://QuickConnect.to/my-rental-analysis`

**注意**：QuickConnect 主要用於訪問 DSM 和 File Station。要直接訪問您部署的租屋分析系統（端口 8000），您需要進行**端口轉發**設定。

### 2.3. 設定端口轉發 (Port Forwarding)

1.  **登入您的路由器**：
    打開瀏覽器，輸入路由器的 IP 位址（通常是 `192.168.1.1` 或 `192.168.0.1`），登入管理介面。

2.  **找到端口轉發設定**：
    這個選項通常在「進階設定」、「防火牆」或「虛擬伺服器」等選單中。

3.  **新增規則**：
    -   **服務名稱**：輸入 `Rental Analysis`。
    -   **通訊協定**：選擇 `TCP`。
    -   **外部端口**：輸入 `8000` (或者您喜歡的其他端口，例如 `8888`)。
    -   **內部 IP 位址**：輸入您 NAS 的內部 IP 位址。
    -   **內部端口**：輸入 `8000`。

4.  **保存並應用**。

設定完成後，您就可以通過您的**公共 IP 位址**或**動態 DNS (DDNS)** 來訪問系統了。

### 2.4. 使用 DDNS (推薦)

由於家庭網絡的公共 IP 位址可能會變動，建議使用 DDNS 服務來獲得一個固定的網址。

1.  **設定 DDNS**：
    -   在 DSM 的「控制台」>「外部存取」>「DDNS」中，可以免費設定一個 Synology 提供的 DDNS。
    -   例如，您可以設定 `my-nas.synology.me`。

2.  **通過 DDNS 訪問**：
    設定好端口轉發和 DDNS 後，您就可以使用以下網址從外部訪問您的系統了：

    ```
    http://my-nas.synology.me:8000/index_v6.html
    ```
