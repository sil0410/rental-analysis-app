# Upload 資料夾

此資料夾存放 CSV 文件，系統啟動時會自動導入。

## CSV 欄位規格

| 欄位名 | 類型 | 必填 | 說明 | 範例 |
|--------|------|------|------|------|
| 案件編號 | 文字 | ✅ | 591 房源唯一 ID | `20377386` |
| 地址 | 文字 | ✅ | 完整地址（含城市+區域） | `新北市中和區景平路123號` |
| 租金 | 整數 | ✅ | 月租金 | `19933` |
| 坪數 | 浮點數 | ✅ | 房源面積 | `16.5` |
| 房型 | 文字 | ⬚ | 房間配置 | `2房` |
| 建築類型 | 文字 | ⬚ | 公寓/電梯大樓 | `公寓` |
| 緯度 | 浮點數 | ✅ | 十進位緯度 | `25.004722` |
| 經度 | 浮點數 | ✅ | 十進位經度 | `121.496389` |
| 週次 | 文字 | ✅ | YYWW 格式 | `2603` |
| 標題 | 文字 | ⬚ | 房源標題 | `近捷運站兩房` |
| 樓層 | 文字 | ⬚ | 樓層資訊 | `4F/5F` |

## 座標轉換

度分秒格式轉十進位：
```
25°0'17"N → 25 + 0/60 + 17/3600 = 25.004722
121°29'47"E → 121 + 29/60 + 47/3600 = 121.496389
```

## 週次格式

格式：`YYWW`（4位數字）
- YY：年份後兩位（如 26 = 2026）
- WW：ISO 週數（01-53）

## 文件命名格式

`{城市}_{區域}_{週次}.csv`

範例：`新北市_中和區_2603.csv`
