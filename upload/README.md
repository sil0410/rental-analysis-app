# Upload 資料夾

請將 CSV 文件上傳到此資料夾。

## CSV 格式要求

CSV 文件需要包含以下欄位：

| 欄位名稱 | 說明 | 範例 |
|---------|------|------|
| 地址 | 房源地址 | 新北市中和區永和路123號 |
| 租金 | 月租金 | 25000 |
| 坪數 | 房屋坪數 | 20.5 |
| 房型 | 房間配置 | 2房 |
| 建築類型 | 公寓/電梯大樓 | 電梯大樓 |
| 緯度 | GPS 緯度 | 25.0088 |
| 經度 | GPS 經度 | 121.4988 |
| 週次 | 數據週次 | 2604 |

## 上傳方式

1. 點擊 `Add file` → `Upload files`
2. 拖拽或選擇您的 CSV 文件
3. 點擊 `Commit changes` 提交

系統會自動檢測並導入新的 CSV 文件。
